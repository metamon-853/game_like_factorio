# game_like_factorio

## ゲームのコンセプトと方向性

### 基本方針

このゲームはFactorio風のゲームですが、Factorioとの差別化を図ることを目指しています。

**一言で表すと：** 「**現実世界の物質をなんでも作れるFactorio風ゲーム**」

### Factorioの特徴

Factorioは以下のような特徴を持つゲームです：
- **自動化**: ベルトコンベアやロボットなどを用いた生産ラインの自動化
- **大量生産**: 効率的な工場設計による大規模な生産システム
- **素材と合成物**: 基本的な素材から複雑な合成物を作り出すシステム
- **最適化**: 限られたレシピでの効率・最適化・圧縮が主な快楽

### 本ゲームの特徴

本ゲームの最大の特徴は、**素材と合成物の種類の圧倒的多さ**です。

- 日常でよく目にする物質やアイテムを、なんでも合成できるようにすることを目標としています
- 現実世界の多様な物質やアイテムをゲーム内で再現し、それらを組み合わせて新しいアイテムを作り出す楽しさを提供します
- Factorioの「最適化」に対して、本ゲームは「**発見・知識・組み合わせの驚き**」を主な快楽とします

### 差別化のポイント

> **Infinite Craft × Factorio × 現実物質図鑑**

Factorioと**プレイヤーが得る快楽の方向性が真逆**であることが最大の強みです。

- **Factorio**: 限られたレシピでの効率・最適化・圧縮
- **本ゲーム**: 無限に近いレシピでの発見・知識・組み合わせの驚き

同じ生産ゲーム文脈にありながら、異なる快楽を提供します。

## 強いポイント

### Factorioの"正反対の快楽"を狙っている

同じ生産ゲーム文脈にありながら、プレイヤーが得る快楽の方向性が真逆なのが強みです。

### コンセプトが一言で伝わる

- 「現実世界の物質をなんでも作れるFactorio風ゲーム」
- 説明一行でワクワクできる
- 理系・クラフト好き・知識欲が強い層に刺さる

---

## 注意点・リスク

### 最大のリスク

> **「何でも作れる」が「何をすればいいか分からない」になる**

- レシピが多いだけだと Infinite Craft 的なおもちゃになりやすい
- Factorio的な目的・手段・達成の構造が弱くなる危険

👉 **探索・発見・体系化そのものがゲームプレイになる設計**が必須
