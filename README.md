# game_like_factorio

## ゲームのコンセプトと方向性

### 基本方針

このゲームはFactorio風のゲームですが、Factorioとの差別化を図ることを目指しています。

**一言で表すと：** 「**現実世界の物質をなんでも作れるFactorio風ゲーム**」

### Factorioの特徴

Factorioは以下のような特徴を持つゲームです：
- **自動化**: ベルトコンベアやロボットなどを用いた生産ラインの自動化
- **大量生産**: 効率的な工場設計による大規模な生産システム
- **素材と合成物**: 基本的な素材から複雑な合成物を作り出すシステム
- **最適化**: 限られたレシピでの効率・最適化・圧縮が主な快楽

### 本ゲームの特徴

本ゲームの最大の特徴は、**素材と合成物の種類の圧倒的多さ**です。

- 日常でよく目にする物質やアイテムを、なんでも合成できるようにすることを目標としています
- 現実世界の多様な物質やアイテムをゲーム内で再現し、それらを組み合わせて新しいアイテムを作り出す楽しさを提供します
- Factorioの「最適化」に対して、本ゲームは「**発見・知識・組み合わせの驚き**」を主な快楽とします
