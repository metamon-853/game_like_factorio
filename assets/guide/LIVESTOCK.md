# 家畜

## Lキー: 家畜を配置 / 製品を収穫

・作物（餌）を持っている状態でLキーを押すと家畜を配置します
・成熟した家畜は一定時間ごとに製品（卵、ミルク、羊毛など）を生産します
・製品が生産されたらLキーで収穫できます
・家畜の種類によって生産間隔が異なります

## Kキー: 家畜を殺して肉を取得

・家畜がいる場所でKキーを押すと家畜を殺して肉を取得できます
・家畜を殺すと、その家畜は消滅します
・肉は重要な食料源となります

# 家畜の文明レベル

・家畜の種類によって必要な文明レベルが異なります
・文明レベルが上がると、より高度な家畜を配置できるようになります
・家畜を育てて製品を生産することで、文明レベルを上げる条件を満たせます
