# インベントリと図鑑

## インベントリ（Eキー）

・インベントリでアイテムをクラフトできます
・必要な素材と要求条件（道具や施設）を満たしていればクラフト可能です
・クラフトしたアイテムはインベントリに追加されます

## アイテム図鑑

・アイテム図鑑でアイテムの詳細を確認できます
・アイテムの説明、必要な素材、要求条件などを確認できます
・カテゴリ別にアイテムを検索できます

---

# セーブとロード

## セーブ/ロード

・ポーズメニュー（ESCキー）からゲームをセーブ/ロードできます
・セーブデータは複数保存可能です
・プレイヤーの位置、インベントリ、文明レベルなどが保存されます

---

# 文明レベル

## 文明レベルについて

・文明レベルが上がると新しいアイテムが利用可能になります
・文明レベル1（旧石器時代）から始まります
・文明レベル5（古代文明時代）まで進行可能です
・文明レベルが上がると、より高度な道具や施設が使用可能になります

## 文明レベル進行条件

・レベル1 → レベル2: パンを100保存、干し肉を50保存
・レベル2 → レベル3: 畜産物を累計20生産
・レベル3 → レベル4: 畜産物を累計100生産
・レベル4 → レベル5: 神殿を1つ以上建設

詳細な進行方法については「エンディング」セクションを参照してください。
